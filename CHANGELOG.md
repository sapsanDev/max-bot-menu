# Changelog

Все заметные изменения в этом проекте будут задокументированы в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
и этот проект придерживается [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-05-07

### Добавлено
- Начальный релиз библиотеки max-menu
- Класс `Menu` для создания интерактивных меню
- Класс `MenuRange` для построения структуры меню
- Полная поддержка различных типов кнопок:
  - Текстовые кнопки (`text()`)
  - Кнопки-ссылки (`url()`)
  - Кнопки открытия приложений (`openApp()`)
  - Кнопки запроса контактов (`contact()`)
  - Кнопки сообщений (`message()`)
  - Кнопки геолокации (`geoLocation()`)
  - Кнопки подменю (`submenu()`)
  - Кнопки возврата (`back()`)
- Динамические кнопки с функцией `dynamic()`
- Система иерархии меню с поддержкой подменю
- Навигация меню (переход, возврат, обновление, закрытие)
- Функция `createSession()` для управления сеансами пользователей
- Функция `resolve()` для работы с динамическими значениями
- Полная документация JSDoc для всех методов
- Примеры использования в README

### Технические детали
- Поддержка TypeScript с полной типизацией
- Middleware интеграция с MAX Bot API
- Поддержка async/await
- Цепочка middleware обработчиков
