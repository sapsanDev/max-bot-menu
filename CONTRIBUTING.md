# Вклад в max-bot-menu

Спасибо за интерес к проекту! Мы приветствуем любые вклады.

## Как начать

1. **Fork** репозиторий
2. **Clone** ваш форк:
   ```bash
   git clone https://github.com/sapsanDev/max-bot-menu.git
   cd max-bot-menu
   ```
3. **Установите зависимости**:
   ```bash
   npm install
   ```
4. **Создайте feature branch**:
   ```bash
   git checkout -b feature/название-функции
   ```

## Процесс разработки

### Сборка

```bash
npm run build
```

Это скомпилирует TypeScript код в `dist/` директорию.

### Правила кодирования

- Используйте TypeScript
- Добавьте JSDoc комментарии для всех публичных методов
- Следуйте существующему стилю кода
- Убедитесь, что код компилируется без ошибок

### Структура проекта

```
src/
├── index.ts      # Главная точка входа, экспорт публичного API
├── menu.ts       # Основные классы Menu и MenuRange
├── session.ts    # Функция createSession
├── types.ts      # TypeScript типы
└── utils.ts      # Вспомогательные функции
```

## Отправка изменений

1. **Commit** ваши изменения:
   ```bash
   git add .
   git commit -m "Описание изменения"
   ```

2. **Push** на ваш форк:
   ```bash
   git push origin feature/название-функции
   ```

3. **Откройте Pull Request**:
   - Дайте четкое название и описание
   - Укажите, какую проблему решает PR
   - Приведите примеры использования новой функции

## Требования для Pull Request

- ✅ Код компилируется без ошибок
- ✅ Все новые публичные методы имеют JSDoc
- ✅ Примеры включены в комментарии
- ✅ Изменения документированы в CHANGELOG.md
- ✅ Нет конфликтов с `main` веткой

## Стиль кода

### Именование

```typescript
// ✅ Хорошо
const getUserMenu = () => {};
const isActive = true;
const MAX_RETRIES = 3;

// ❌ Плохо
const get_user_menu = () => {};
const IsActive = true;
const max_retries = 3;
```

### JSDoc комментарии

```typescript
/**
 * Описание функции
 * @param name - Описание параметра
 * @returns Описание возвращаемого значения
 * @example
 * // Пример использования
 * myFunction('value');
 */
function myFunction(name: string): void {
  // реализация
}
```

### Типизация

```typescript
// ✅ Хорошо - явная типизация
const menu: Menu<Context> = new Menu('main');
const handler: Middleware<Context> = async (ctx, next) => {};

// ❌ Плохо - неявная типизация
const menu = new Menu('main');
const handler = async (ctx, next) => {};
```
